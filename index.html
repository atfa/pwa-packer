Step 1 复制指令 (Prompt)

把这段话发给 Gemini，让它生成包含嵌入文件的 HTML 代码： 

我需要你帮我把这个网页升级为 PWA（Progressive Web App）。
需求列表：
主要功能： 保持原有的网页设计和功能不变，变为可直接部署的静态网页。
PWA 注册逻辑： 在 <head> 或 <body> 底部加入常规的 Service Worker 注册代码，以及 <link rel="manifest" href="manifest.json"> 标签。
特殊文件嵌入方式（关键）：
请不要将 manifest.json 和 sw.js 分别输出，而是将它们的内容作为数据块直接嵌入在 HTML 的 </body> 标签之前。
请严格使用以下 <script> 标签格式包裹它们，以确保页面不报错且不显示：

xml
<!-- 嵌入 Manifest 文件 -->
<script type="text/file" data-filename="manifest.json">
{
  "name": "App Name",
  "short_name": "App",
  "start_url": "./index.html",
  ...
}
</script>
<!-- 嵌入 Service Worker 文件 -->
<script type="text/file" data-filename="sw.js">
const CACHE_NAME = 'v1';
// ... sw code here ...
</script>

请注意： 
1. 确保 <script> 标签的 type 属性必须是 text/file，并使用 data-filename 属性标记文件名。
2. HTML文件名为index.html，所以manifest.json 中的 "start_url" 必须设置为 "./index.html"，sw.js中也要用到index.html。
