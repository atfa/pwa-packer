API Key 安全接入助手

Prompt 驱动开发 · 一键打包部署

工作原理

此工具不涉及复杂的 PWA 拆分。它会将您粘贴的代码直接保存为 index.html 并打包成 ZIP，方便您直接拖入 Netlify 进行更新或部署。

Step 1 复制改造指令 (Prompt)

发送给 Gemini/ChatGPT，让它帮你把 API Key 逻辑加进去： 

请帮我改造这个代码，实现下面的需求：

1. API 密钥不能写在代码里。
2. 页面加载时检查本地存储 (localStorage)：
   - 如果本地没有密钥：显示一个输入框让用户输入密钥，并提供“保存”按钮。保存后将密钥存入 localStorage 并刷新页面。
   - 如果本地存在密钥：则自动读取密钥，隐藏输入框，直接进入应用功能。
3. 在页面右上角加一个"修改密钥"的按钮。点击后清除本地存储并重新加载。
4. 样式要求：输入框、按钮的风格需要与原来的页面风格保持一致。
5. **保持原来的核心功能和界面布局不变**，只增加密钥管理逻辑。
